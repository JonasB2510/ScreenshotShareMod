## 🇩🇪

Klicke auf den gelben Text um einen Hochlade-Dialog zu öffnen
![Tutorial Image 01](images/01-screenshotclick.png?raw=true "Tutorial 01")

Ein hochlade-Dialog in dem du auswählen kannst ob du deine Aufnahme hochladen willst
![Tutorial Image 02](images/02-uploaddialog.png?raw=true "Tutorial 02")

Wenn du auf diesen gelben Text klickst bekommst du einen Befehl in deine Befehlszeile.
![Tutorial Image 03](images/03-sharebutton.png?raw=true "Tutorial 03")

Drücke ENTER zum senden.
![Tutorial Image 04](images/04-sendshare.png?raw=true "Tutorial 04")

Jeder mit dieser Mod wird den Text bei sich so angezeigt bekommen und kann anschließend darauf drücken um die Aufnahme zu öffnen.
![Tutorial Image 05](images/05-sharechat.png?raw=true "Tutorial 05")
